# Open Source Project LAB 3
동의대학교 오픈소스 프로젝트 교과목 LAB 3 과제에 대한 레포지토리입니다.

## 실행 방법
### 1. git clone
``` bash
git clone https://github.com/DAADAISMYLIFE/open_source_LAB3.git
```

### 2. 원하는 챕터로 이동
``` bash
cd 4_type
```

### 3. 실행파일 실행
```bash름
./<실행파일 이름>
```

#### 주의사항
챕터 1 rust는 실행 방법이 다름
```bash
cd 1_rust

cargo build

cargo run
```
